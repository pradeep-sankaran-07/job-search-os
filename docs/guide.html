The Job Search OS

in Claude Code

A 15-minute, one-time setup by talking to Claude Code. After that, every morning your job tracker is updated with new roles, cover letters are drafted, and you spend 20 minutes reviewing instead of three hours browsing.

The problem

Manual job search burns hours.

Same searches, different tabs. Rewritten cover letters. Dead links. Roles you already saw. The signal is buried in noise.

What it does

Talk to Claude Code. It does the rest.

Searches the main job boards daily against your profile. Drafts a first-draft cover letter for every strong fit. The tracker updates itself.

Install and use

Install Claude Code, open it, and tell it to install the plugin from the URL below. Claude will ask a few short questions — that is the setup.

github.com/pradeep-sankaran-07/job-search-os
Works on macOS, Linux, and Windows. Needs Claude Pro or Max and the Chrome browser (with the claude-in-chrome extension, for LinkedIn searches).
Four steps, all in plain English
1

Drop your CV

Claude reads it, offers to polish it, saves both versions for you.

“Here’s my CV — please clean it up and save it for my job search.”
2

Build your profile

A few short questions about titles, locations, seniority, and what to avoid.

“Help me set up my job-search profile.”
3

Pick target companies

One quick handoff to Claude.ai Research, then back to Claude Code.

“Help me build my list of target companies.”
4

Your daily run

Overnight search, live-verified matches, draft cover letters in your tracker.

“What’s new today?”
