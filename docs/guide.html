Your morning after setup

Open Claude Code. Type /job-search.

  • New roles fetched overnight from Finn, LinkedIn, Indeed, Glassdoor

  • Cover letter pre-drafted for every strong fit, in your voice
  • Outreach note drafted for a LinkedIn referral before you apply
The problem

Manual job search burns hours

Finn.no, LinkedIn, Indeed — the same search run three different ways every morning. Then rewriting the same cover letter. Dead links. Roles you already saw. Cover letters that sound like ChatGPT. The signal is buried in noise.
What this does

One plugin, four sources, every morning

A Claude Code plugin searches the major Norwegian + European boards every day, against your profile. Every match is verified live in a real browser. Every strong fit gets a first-draft cover letter in your voice. The tracker updates itself.

Install (3 commands)

  1. Install Claude Code: npm i -g @anthropic-ai/claude-code
  2. In Claude Code, install the plugin (right →)
  3. Run /job-search-setup and follow the wizard.
Requires: Claude Pro or Max · Node.js · Chrome (with the claude-in-chrome extension, for LinkedIn). The setup wizard detects your OS (macOS, Linux, Windows) and installs the rest silently.
